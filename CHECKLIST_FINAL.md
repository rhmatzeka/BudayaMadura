```

   link_youtube = "https://www.youtube.com/watch?v=..."

2. Model automatically extracts ID

3. Model automatically generates embed URL
   embed_url = "https://www.youtube.com/embed/dQw4w9WgXcQ"

4. Template renders embedded player

Result: YouTube video embedded in page
```
